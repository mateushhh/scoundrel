# Windows Build

This branch contains the compiled, playable Windows version of the game.

## How to Play

1. Click the green **<> Code** button at the top right of this page.
2. Select **Download ZIP**.
3. Extract the downloaded ZIP file.
4. Open the `build` folder.
5. Double-click the `.exe` file to start the game.
